# 《存在的天平（Find-Balance）》

> 英文名：**find balance**
> 一套希望自己活得更好的生活哲学。  


## 📖 关于本书

我们每个人都是一架天平：左边是精神世界（思想、意义、直觉），右边是物质世界（身体、环境、行动）。当天平倾斜时，我们会陷入内耗或麻木。或许可以尝试找到属于自己的平衡，活得舒展而充实。

本书将带你：
- 介绍一种新的认识自己的方式
- 尝试用“三个爱好”让生活更充实
- 掌握日常的“复位急救包”和“日记反思法”
- 或许我的生活小哲学可以帮助到你，给你启发
